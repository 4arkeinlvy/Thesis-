# Kuesioner Evaluasi Usabilitas Aplikasi SmartCityApps

**Instrumen:** System Usability Scale (SUS) — Brooke (1996); terjemahan Bahasa Indonesia tervalidasi Sharfina & Santoso (2016).
**Responden:** 15 Warga Pelapor (pengguna smartphone Android).
**Catatan pengisian Google Form:** Bagian D paling cocok memakai tipe **"Grid pilihan ganda" (multiple choice grid)** atau **"Skala linier 1–5"** per pernyataan. Bagian E memakai **"Paragraf"**.

---

## Pembuka (deskripsi form)

Terima kasih telah bersedia mencoba aplikasi **SmartCityApps**, aplikasi pelaporan masalah lingkungan kota berbasis kecerdasan buatan. ## Bagian D — Pernyataan SUS (skala 1–5)

**Skala:** 1 = Sangat Tidak Setuju · 2 = Tidak Setuju · 3 = Netral · 4 = Setuju · 5 = Sangat Setuju

| No. | Pernyataan |
|-----|-----------|
| P1  | Saya berpikir akan menggunakan aplikasi ini lagi. |
| P2  | Saya merasa aplikasi ini rumit untuk digunakan. |
| P3  | Saya merasa aplikasi ini mudah digunakan. |
| P4  | Saya membutuhkan bantuan dari orang lain atau teknisi dalam menggunakan aplikasi ini. |
| P5  | Saya merasa fitur-fitur aplikasi ini berjalan dengan semestinya. |
| P6  | Saya merasa ada banyak hal yang tidak konsisten (tidak serasi) pada aplikasi ini. |
| P7  | Saya merasa orang lain akan memahami cara menggunakan aplikasi ini dengan cepat. |
| P8  | Saya merasa aplikasi ini membingungkan. |
| P9  | Saya merasa tidak ada hambatan dalam menggunakan aplikasi ini. |
| P10 | Saya perlu membiasakan diri terlebih dahulu sebelum menggunakan aplikasi ini. |

> **Jangan diubah urutannya.** P ganjil = pernyataan positif, P genap = pernyataan negatif. Pola selang-seling ini wajib untuk validitas SUS.

## Bagian E — Umpan Balik Kualitatif (isian paragraf)

1. Menurut Anda, apa kelebihan utama aplikasi SmartCityApps?
2. Bagian atau fitur apa yang menurut Anda masih membingungkan atau perlu diperbaiki?
3. Apakah Anda bersedia menggunakan aplikasi ini untuk melaporkan masalah lingkungan di sekitar Anda? Mohon jelaskan alasannya.

---

## Cara menghitung skor SUS (untuk rekap, bukan ditampilkan ke responden)

Untuk tiap responden:
- Pernyataan **ganjil** (P1, P3, P5, P7, P9): kontribusi = (jawaban − 1)
- Pernyataan **genap** (P2, P4, P6, P8, P10): kontribusi = (5 − jawaban)
- **Skor SUS** = (jumlah seluruh kontribusi) × 2,5  → rentang 0–100

Interpretasi (Bangor dkk., 2009): >68 di atas rata-rata · 68–80 baik · 80–90 sangat baik · >90 luar biasa · <68 perlu perbaikan.
